# AGENTS.md — commandes & conventions

## Setup
- Installer : npm ci
- Dev : npm run dev
- Build : npm run build
- Tests : npm test
- Lint : npm run lint

## Règles
- Ne pas créer/commiter de fichiers binaires lourds.
- Utiliser des placeholders ou charger les assets au build.

## Avant chaque PR
- npm run lint && npm test
